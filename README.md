Interactive browser-based generative particle artwork inspired by underwater organisms, digital matter, procedural dot systems, and Houdini-style parametric animation.

## What this project does

Digital Underwater Organism creates a living animated structure directly in the browser.  
The artwork is generated from thousands of digital dots, short connecting lines, halo particles, protruding fibers, and procedural body articles.

The visual system behaves like a synthetic aquatic creature: a central dotted body with moving fibers, a long expressive tail, head protrusions, halo particles, and soft organic motion. The structure can be reshaped in real time through sliders, mode buttons, randomization, and export tools.

## Why this project is useful

This project is useful as:

- a browser-based generative artwork
- a mobile-friendly interactive visual app
- a procedural art prototype
- a study of digital organisms made from particles
- a lightweight alternative to rendered animation previews
- a visual experiment for exhibition, social media, or creative coding portfolios

Instead of producing one fixed image, the app keeps the creature alive through animation, parameter changes, randomized variations, and export options.

## Controls

Use the bottom transparent control panel to reshape the organism.

Main actions:

- **Randomize** — creates a new organism variation
- **Export PNG** — saves the current frame as a still image
- **Export .MP4** — records a 15-second animation sequence from the canvas

Display modes:

- **Hybrid** — combines dots and connecting lines
- **Dots** — shows the organism as a pure particle system
- **Fine** — creates a lighter, more delicate dotted structure

Important sliders:

- **Digital Grid** — controls how strongly the dots snap to a digital grid
- **Scale** — changes the size of the whole organism, up to 300
- **Center X / Center Y** — moves the composition on the screen
- **Head Position** — shifts the position of head protrusions
- **Tail Position** — shifts the position of the tail fibers
- **Body Articles** — changes the number of segmented body sections
- **Dots Per Article** — changes density across the body
- **Fiber Amount** — controls how many fibers grow from the body
- **Fiber Dots** — changes how detailed each fiber path becomes
- **Body Thickness** — changes the width of the central body
- **Fin Spread** — controls how far the side fibers extend
- **Asymmetry** — makes the organism more irregular and organic
- **Movement** — changes the animation speed
- **Curl** — controls the curvature and wave motion of fibers
- **Halo Dots** — controls the amount of surrounding digital particles
- **Dot Size** — changes the thickness of the particle points
- **Brightness** — controls overall particle brightness
- **Contrast** — increases or softens the black-and-white contrast
- **Tilt** — rotates the organism

## Exporting images and video

### Export PNG

Click **Export PNG** to save the current canvas frame as an image.

This is useful for:

- still artworks
- Instagram posts
- presentation images
- thumbnails
- visual documentation

### Export .MP4

Click **Export .MP4** to record a 15-second animation sequence.

The app attempts to export MP4 first.  
If the browser does not support MP4 canvas recording, it automatically falls back to WEBM.

For best results:

- use Chrome, Edge, or Safari
- keep the browser tab active during recording
- wait until the download starts
- reduce particle density if your device becomes slow

## Mobile use

The interface is designed for phone screens.

The bottom panel stays transparent so the organism remains visible behind the controls.  
The parameters are arranged in two columns to make the menu shorter and easier to use on mobile.

If performance becomes slow on a phone:

- reduce **Fiber Amount**
- reduce **Fiber Dots**
- reduce **Halo Dots**
- reduce **Dots Per Article**
- reduce **Scale**
- switch to **Fine** mode

## Creative concept

The artwork explores a hybrid between:

- underwater life
- procedural geometry
- synthetic biology
- particle simulation
- digital fabric
- generative drawing
- computational organism design

The creature is not modeled as a traditional 3D mesh.  
It is constructed from animated points, short lines, particles, and procedural movement, giving it a semi-organic and semi-digital presence.
